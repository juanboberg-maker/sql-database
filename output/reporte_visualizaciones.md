# Informe de visualización y análisis Spotify

## Tablas utilizadas
- albums
- genre
- Audio_Features
- Audio_Analysis

## Visualizaciones generadas
- Top artistas a partir de albums.
- Top géneros a partir de genre.
- Distribución por categoría de tempo a partir de Audio_Analysis.
- Comparación de energía alta vs media/baja desde Audio_Analysis.
- Promedio de audio features desde Audio_Features.
- Popularidad media por género mediante join entre albums y genre.

## Insights
- El artista con más canciones es George Jones con 260 registros.
- El género con más canciones es acoustic con 1000 registros.
- La categoría de tempo predominante es Rápido con 47755 canciones.
- El género con mayor popularidad media es k-pop con 59.36 de popularidad media.
